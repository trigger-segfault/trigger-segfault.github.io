Change Log
Version 1.0.2 beta

Hotfix 1.0.2.6

• Fixed a bug with the santa drops autobuying.
• Autobuy no longer puts priority of research and seasonal items over buildings and upgrades if it can't afford them.

Hotfix 1.0.2.5

• Fixed a problem that prevented AutoCookie from loading properly if an autoclicker was on.
• Re-implemented Golden Cookie Notifications.

Hotfix 1.0.2.4

• All mod tabs update every 1 second instead of 5 seconds.
• Valued upgrades will no longer have a BCI of zero.
• The end season upgrade will now always be ordered before the season upgrades.
• BCI is not rounded, thus buildings will no longer be all green or red after when their BCI is smaller than one.
• Added a change log button to view the website from the mod menu. (Don't worry, it opens in a new tab.)

Hotfix 1.0.2.3

• Fixed a problem causing AutoCookie to buy seasons until it couldn't afford them.

Hotfix 1.0.2.2

• Fixed a problem where autobuy would stop working.
• Now says mod version below Trigger Cookies.

Hotfix 1.0.2.1

• Fixed issues caused by accidental revert.

Version 1.0.2.0

• Introduced upgrade efficiency coloring.
• Building and upgrade BCIs now show in their tooltip.
• You now have the option to prevent the Grandmapocalypse when autobuying research.
• LayoutCookie renamed to EnhanceCookie.
• Added a shortened numbers options.
Version 1.0.1 beta

Version 1.0.1.0

• Added the ability to save mod settings.
• Finished modular support. Custom mods can now be added.
• Moved repository over to GitHub pages to prevent mime type errors.
• LayoutCookie renamed to EnhanceCookie.
• Added settings to handle selling Chocolate egg on ascend.
• Added building efficiency coloring.
• Added Golden Cookie and Wrinkler stats.
Version 1.0.0 beta

Version 1.0.0.0

• Released TriggerCookies to the public.
• Includes AutoCookie, LayoutCookie, StatCookie, HotfixCookie, and CheatCookie.
• No saving settings yet.

Features
Author: Robert Jordan
Version: v1.0.1.1
RCT2 Audio Extractor is a simple tool for extracting all sounds and music in RollerCoaster Tycoon 2 from the css files. All the sounds and music are output into the specified directory in wave format and MP3 format for music if specified. If music is saved as MP3 then songs will also be given metadata on the artists, title, and track number. To find your css files go to your RCT2 directory and select your Data folder.

Extracting
Controls
Extract
Begins extracting sounds from the css files to the specified directory.
Data Directory
Select the directory containing css#.dat files. This should be the Data folder in your RCT2 directory.
Output Directory
Select the directory to output the sounds and music to.
Extract Music As MP3
All music will be extracted as MP3's instead of wave files. All music will be given metadata about the song.
